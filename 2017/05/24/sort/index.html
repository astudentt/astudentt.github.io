选择排序

Posted on 2017-05-24 | Edited on 2019-02-27

排名是我们经常遇到的问题，小的时候成绩排名，身高排名，各种排名。听歌曲时有各种各样的排行榜，那么如果让你做一个排行榜，你要怎样把杂乱无章的数据，变成人们能够准确判断的排行榜呢？那么你就需要用到排序算法了，排序算法有很多，选择排序、快速排序、冒泡排序等，此文章介绍的是简单易懂的选择排序法。选择排序是一种灵巧的排序方法，但是速度并不是很快，它的时间复杂度为O（n 2）

  1. 选择排序说明

选择排序的主要思想是：

  1. 声明一个新的数组作为排序好的数组的容器。
  2. 每次对现有数组进行最大值或最小值查找（根据需要），将查找到的值顺序添加到新数组中，并将其从原数组中剔除。

  3. 重复第二步骤直到原数组的长度为一，即可停止，最后得到的新数组则是排序好的数组。

  4. 举例说明

1）音乐播放器都会有很多统一或自己的排行榜，他就需要根据音乐的播放量或者唱片销量进行排序，然后对应显示。

  1. 代码说明

下面我们先写一个用于查找最小值的方法：
function findSmallest(arr){
var smallest = arr[0];
var smallest_index = 0;
for(var i=0;i arr[i]){
smallest = arr[i];
smallest_index=i;
}
}
return smallest_index;
}
利用findSmallest方法实现选择排序方法：
var newArr = [];
function selectionSort(arr){
if(arr.length >0){
smallest = findSmallest(arr);
newArr.push(arr[smallest]);
arr.splice(smallest,1);
selectionSort(arr);
}
return newArr;
}
var arr = [3,5,7,9,22,4,66,2,77];
console.log(selectionSort(arr));

  1. 结论

选择排序的方法是非常容易理解的，在数据量不大的情况下，可以使用，如果对效率有着严格要求的话，建议使用时间复杂度更低的算法。
